# Nested if Statements
You can use one **if-else** statement inside another **if** or **else** statement.  
**For example:**

```java
int age = 25;
if(age > 0) {
	if(age > 16){
		System.out.println("Welcome!");
	} else{
		System.out.println("Too Young");
	}
} else {
	System.out.println("Error");
}
```
